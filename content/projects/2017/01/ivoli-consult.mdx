# Project overview

Ivoli Consult was a web development project that aimed to create a website for an accountant. This project primarily
involved front-end web technologies, including JavaScript, CSS, and Bootstrap. Unfortunately, the website is no longer
live, but it served as an online platform for the accountant's services.

# Key contributions

- Worked closely with the client to understand their specific requirements and objectives.
- Developed a professional website to represent the services and expertise of the accountant.
- Utilized front-end technologies such as JavaScript and CSS to create a visually appealing and user-friendly design.
- Integrated Bootstrap to ensure a responsive and mobile-friendly user experience.

# Project impact

The Ivoli Consult website provided the accountant with an online presence, enabling them to showcase their services to
potential clients. While the website is no longer live, it once served as an important tool for promoting the
accountant's professional services.
