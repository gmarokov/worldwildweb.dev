# Project overview

Hanay.eu/bg was a web development project focused on creating a static website for a local restaurant. The project
involved front-end web technologies, including JavaScript, CSS, and Bootstrap. The website aimed to provide an online
presence for the restaurant and promote its offerings.

# Key contributions

- Worked closely with the client to understand their specific requirements and objectives.
- Developed a static website for Hanay, a local restaurant, to showcase its menu, location, and contact information.
- Utilized JavaScript, CSS, and Bootstrap to create an attractive and user-friendly design.
- Ensured the website was responsive and accessible to a wide range of users.

# Project impact

The Hanay.eu/bg website served as a valuable online platform for the local restaurant. It provided essential information
to potential customers, such as the restaurant's menu and location. While the project is static and no longer live, it
once played a crucial role in enhancing the restaurant's online presence.

View the website at [Hanay.bg](https://hanay.bg)
