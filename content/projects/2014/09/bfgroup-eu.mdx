# Project overview

I had the privilege of working on the BFGroup.eu project, a website dedicated to the Bulgarian Financial Group. The goal
was to create an informative and user-friendly platform for the financial services sector. This project showcased my
skills in web development, leveraging technologies such as WordPress, PHP, MySQL, JavaScript, CSS, and Bootstrap.

# Key contributions

- Designed and developed the website to meet the client's requirements and expectations.
- Implemented WordPress as the content management system, providing an easy-to-manage solution for the client.
- Utilized PHP, MySQL, JavaScript, CSS, and Bootstrap to create a responsive and visually appealing website.
- Ensured the website's functionality and performance, enhancing the user experience.

# Project impact

The BFGroup.eu website served as a valuable online presence for the Bulgarian Financial Group, offering clients and
visitors a comprehensive resource for financial services information. The project highlighted my proficiency in web
development and my ability to meet client objectives.

View the website at [BFGroup.eu](https://bfgroup.eu/)
